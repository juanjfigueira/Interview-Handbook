# Interview Handbook

> A structured interview preparation project for becoming a confident Senior Data Engineer in English.

---

# Overview

This repository contains my personal interview preparation handbook.

The objective is not to memorize answers.

The objective is to communicate naturally, confidently, and professionally during technical interviews conducted in English.

Every chapter is based on my own professional experience as a Senior Data Engineer and has been designed to prepare me for interviews across Portugal and Europe.

---

# Main Goal

By the end of this training I should be able to:

- Introduce myself confidently in English.
- Explain every project on my CV naturally.
- Discuss technical topics without translating from Spanish.
- Demonstrate senior-level communication skills.
- Successfully complete HR, technical and system design interviews.

---

# Training Philosophy

This handbook is based on one simple principle:

> **Don't study English. Train for interviews.**

Instead of learning random grammar topics, every chapter focuses on real interview situations.

The goal is to build automatic responses through repetition and practice.

# Study Plan

The handbook is divided into two phases.

## Phase 1 — Build the Foundation

**Duration:** 16 days

One chapter per day.

For every chapter:

- Read carefully.
- Understand the concepts.
- Learn the vocabulary.
- Rewrite the answers using my own words.
- Practice speaking aloud.
- Record my answers.
- Improve grammar and pronunciation.

The goal is understanding.

Not memorization.

---

## Phase 2 — Intensive Practice

**Duration:** Remaining days of the month.

Daily practice includes:

- Random HR questions.
- Random technical questions.
- Behavioral questions.
- Architecture interviews.
- Mock interviews.
- Vocabulary review.
- Speaking only in English.

The goal is automatic communication.

---

# Daily Routine

Every training session follows the same workflow.

1. Read the chapter.
2. Highlight new vocabulary.
3. Rewrite the model answers.
4. Practice speaking aloud.
5. Record the answers.
6. Review mistakes.
7. Repeat.

---

# Interview Principles

During every interview:

- Tell stories.
- Explain business impact.
- Be concise.
- Speak naturally.
- Focus on value.
- Think before answering.
- Avoid translating mentally.

---

# Success Criteria

At the end of this project I should be able to:

- Speak confidently about every company on my CV.
- Explain my technical decisions clearly.
- Handle follow-up questions naturally.
- Complete a one-hour interview entirely in English.
- Feel comfortable interviewing for Senior Data Engineer positions anywhere in Europe.
